## Microsoft Engage Project

### Project Structure

    - Chat Room : Application for the chatroom
    - Frontend : Contains the client side pages and components 

### How to run ?

    - Be in your root folder
    - npm install
    - nodemon server
    - cd frontend && npm install  
    - npm start
